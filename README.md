# Palm Madison Apartments HOA Website

This repository contains a Next.js 14 App Router website for the Palm Madison Apartments HOA.

## Pages

- Home (`/`)
- Payments (`/payments`)
- Announcements (`/announcements`)
- Documents (`/documents`)
- Board (`/board`)
- Contact (`/contact`)

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to view the site.

## Build and Lint

```bash
npm run lint
npm run build
```
